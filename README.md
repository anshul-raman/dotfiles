# dotfiles

## install oh-my-zsh 
  > optional: install pk10-theme 

## Create symlinks 
  - ./nvim -> ~/.config/nvim  
  - ./.tmux.conf -> ~/.tmux.conf
  - ./.zshrc -> ~/.zshrc


### TMUX setup 

  - Clone TPM 

> ```bash
> $ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
> ```

  - Install Plugins 
  > `prefix` + `I`
